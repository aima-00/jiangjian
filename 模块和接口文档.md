# 多用户聊天系统模块和接口文档

## 1. 服务器端模块和接口

### 1.1 连接管理模块 (ConnectionManager)

#### 功能描述
负责管理所有客户端连接，包括连接建立、断开、心跳检测等。

#### 接口定义
```cpp
class ConnectionManager {
public:
    // 添加客户端连接
    bool addClient(int socket_fd, const std::string& username);
    
    // 移除客户端连接
    bool removeClient(int socket_fd);
    
    // 获取所有连接的客户端列表
    std::vector<int> getClientSockets();
    
    // 检查客户端是否仍然连接
    bool isClientConnected(int socket_fd);
    
    // 发送心跳包
    void sendHeartbeat(int socket_fd);
};
```

### 1.2 消息路由模块 (MessageRouter)

#### 功能描述
负责处理消息分发和路由，包括单播、组播、广播消息。

#### 接口定义
```cpp
class MessageRouter {
public:
    // 发送单播消息
    bool sendUnicastMessage(int sender_fd, int receiver_fd, const Message& msg);
    
    // 发送组播消息到群组
    bool sendMulticastMessage(int sender_fd, int group_id, const Message& msg);
    
    // 发送广播消息到所有用户
    bool sendBroadcastMessage(int sender_fd, const Message& msg);
    
    // 路由消息到目标用户
    bool routeMessage(const Message& msg);
};
```

### 1.3 用户认证模块 (AuthenticationService)

#### 功能描述
处理用户注册、登录验证和会话管理。

#### 接口定义
```cpp
class AuthenticationService {
public:
    // 用户注册
    AuthResult registerUser(const std::string& username, const std::string& password);
    
    // 用户登录
    AuthResult authenticateUser(const std::string& username, const std::string& password);
    
    // 验证会话令牌
    bool validateSession(const std::string& session_token);
    
    // 生成会话令牌
    std::string generateSessionToken(int user_id);
    
    // 获取用户信息
    UserInfo getUserInfo(int user_id);
};
```

### 1.4 多媒体处理模块 (MediaProcessor)

#### 功能描述
处理语音、图片、视频数据的编码、解码和传输。

#### 接口定义
```cpp
class MediaProcessor {
public:
    // 处理音频数据
    MediaProcessResult processAudio(const AudioData& audio_data, int quality_level = 3);
    
    // 处理图片数据
    MediaProcessResult processImage(const ImageData& image_data, int quality_level = 80);
    
    // 处理视频数据
    MediaProcessResult processVideo(const VideoData& video_data, int quality_level = 23);
    
    // 压缩多媒体数据
    std::vector<uint8_t> compressMedia(const std::vector<uint8_t>& raw_data, MediaType type);
    
    // 解压多媒体数据
    std::vector<uint8_t> decompressMedia(const std::vector<uint8_t>& compressed_data, MediaType type);
};
```

### 1.5 数据持久化模块 (DataPersistence)

#### 功能描述
负责用户数据、消息历史和其他持久化数据的存储和检索。

#### 接口定义
```cpp
class DataPersistence {
public:
    // 保存用户信息
    bool saveUser(const UserInfo& user);
    
    // 加载用户信息
    UserInfo loadUser(int user_id);
    
    // 检查用户名是否唯一
    bool isUsernameUnique(const std::string& username);

    bool saveGroup(const GroupInfo& group);
    
    // 加载群组信息
    GroupInfo loadGroup(int group_id);
    
    // 保存消息历史
    bool saveMessageHistory(const Message& msg);
    
    // 加载消息历史
    std::vector<Message> loadMessageHistory(int group_id, int limit = 100);
};
```

## 2. 客户端模块和接口

### 2.1 UI界面模块 (UIManager)

#### 功能描述
提供用户交互界面，包括登录界面、聊天窗口、联系人列表等。

#### 接口定义
```cpp
class UIManager : public QObject {
    Q_OBJECT
    
public:
    explicit UIManager(QWidget *parent = nullptr);
    
    // 显示登录界面
    void showLoginDialog();
    
    // 显示主聊天窗口
    void showMainChatWindow();
    
    // 显示群组聊天界面
    void showGroupChatWindow(int group_id);
    
    // 添加消息到聊天界面
    void addMessageToChat(int group_id, const Message& msg);
    
    // 更新用户在线状态
    void updateUserStatus(int user_id, UserStatus status);
    
signals:
    // 登录请求信号
    void loginRequested(const QString& username, const QString& password);
    
    // 发送消息信号
    void sendMessageRequested(int group_id, const QString& message);
    
    // 创建群组信号
    void createGroupRequested(const QString& group_name);
};
```

### 2.2 网络通信模块 (NetworkClient)

#### 功能描述
负责与服务器的通信，包括消息发送和接收。

#### 接口定义
```cpp
class NetworkClient : public QObject {
    Q_OBJECT
    
public:
    explicit NetworkClient(QObject *parent = nullptr);
    
    // 连接到服务器
    bool connectToServer(const QString& host, quint16 port);
    
    // 断开与服务器的连接
    void disconnectFromServer();
    
    // 发送登录请求
    bool sendLoginRequest(const QString& username, const QString& password);
    
    // 发送注册请求
    bool sendRegisterRequest(const QString& username, const QString& password);
    
    // 发送消息
    bool sendMessage(int group_id, const QString& message);
    
    // 发送多媒体消息
    bool sendMediaMessage(int group_id, const QByteArray& media_data, MediaType type);
    
signals:
    // 连接状态改变信号
    void connectionStateChanged(bool connected);
    
    // 接收到消息信号
    void messageReceived(const Message& msg);
    
    // 用户登录结果信号
    void loginResult(bool success, const QString& error_message = QString());
    
    // 用户状态更新信号
    void userStatusChanged(int user_id, UserStatus status);
};
```

## 3. 数据结构定义

### 3.1 消息结构
```cpp
struct Message {
    int id;                    // 消息ID
    int sender_id;             // 发送者ID
    int receiver_id;           // 接收者ID（0表示群组消息）
    int group_id;              // 群组ID
    MessageType type;          // 消息类型
    std::string content;       // 消息内容
    std::vector<uint8_t> data; // 多媒体数据
    uint64_t timestamp;        // 时间戳
    std::string session_token; // 会话令牌
};
```

### 3.2 用户信息结构
```cpp
struct UserInfo {
    int id;                    // 用户ID
    std::string username;      // 用户名
    std::string password_hash; // 密码哈希
    UserStatus status;         // 用户状态
    uint64_t last_login_time;  // 最后登录时间
    std::string avatar_url;    // 头像URL
};
```

### 3.3 群组信息结构
```cpp
struct GroupInfo {
    int id;                          // 群组ID
    std::string name;                // 群组名称
    int owner_id;                    // 群主ID
    std::vector<int> member_ids;     // 成员ID列表
    int max_members;                 // 最大成员数（默认8人）
    uint64_t created_time;           // 创建时间
    std::string description;         // 群组描述
};
```

## 4. 消息协议定义

### 4.1 消息头格式
```
[消息类型(1字节)][消息长度(4字节)][序列号(4字节)][保留字段(2字节)]
```

### 4.2 认证相关消息
- `LOGIN_REQUEST`: 用户登录请求
- `LOGIN_RESPONSE`: 登录响应
- `REGISTER_REQUEST`: 用户注册请求
- `REGISTER_RESPONSE`: 注册响应
- `SESSION_VALIDATE`: 会话验证

### 4.3 聊天相关消息
- `TEXT_MESSAGE`: 文本消息
- `AUDIO_MESSAGE`: 音频消息
- `IMAGE_MESSAGE`: 图片消息
- `VIDEO_MESSAGE`: 视频消息
- `GROUP_CREATE`: 创建群组
- `GROUP_JOIN`: 加入群组
- `GROUP_LEAVE`: 离开群组

### 4.4 系统相关消息
- `USER_STATUS_UPDATE`: 用户状态更新
- `HEARTBEAT`: 心跳包
- `ACKNOWLEDGE`: 确认消息
- `ERROR_RESPONSE`: 错误响应
```
